## Landing Page para tienda de ropa

Diseño moderno y llamativo siguiendo la paleta de colores de la empresa

* Se utilizó el framework Astro por su sencillez y rapidez, ideal para un proyecto como este.
* Se integró la API de WhatsApp Business para dirigir a los clientes al canal de ventas.
* Se realizaron wireframes para presentar un diseño inicial de la página y herramientas de IA como Google Stitch para previsualizar el diseño final.
  
Link a la pagina: https://drillmasters.netlify.app/

## Landing Page for a Clothing Store

A modern, eye-catching design that follows the company's color palette

* The Astro framework was used for its simplicity and speed, making it ideal for a project like this.
* The WhatsApp Business API was integrated to direct customers to the sales channel.
* Wireframes were created to present an initial page design, and AI tools such as Google Stitch were used to preview the final design.
  
Link to the page: https://drillmasters.netlify.app/
